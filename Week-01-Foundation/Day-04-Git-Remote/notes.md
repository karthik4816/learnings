# Day 4 - Remote, Clone & Pull

## What is a Remote Repository?

A remote repository is a Git repository hosted on GitHub or another server. It allows developers to collaborate and back up their code.

## git remote -v

 - Displays the remote repository URL.

## git clone

Downloads a complete copy of a remote repository to your local computer.

Example:

-- git clone <repository-url>

## git pull

--Downloads and merges the latest changes from the remote repository into the current branch.

Example:

git pull

## Difference

- git clone: Used once to download a repository.
- git pull: Used regularly to update an existing repository.
